# Google Pay UX Case Study

## Overview
A beginner fintech UX case study designed using Figma focused on student expense tracking and QR fraud detection.

## Features
- Expense tracking
- Spending streak system
- QR fraud warning popup

## Tools Used
- Figma
- Notion

## Goal
To explore smarter and safer digital payment experiences for students.
